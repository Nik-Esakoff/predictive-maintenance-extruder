# Анализ данных экструзии полипропилена

## Описание
Анализ промышленных данных процесса экструзии полипропилена:
- 1.5 млн записей, 43 технологических параметра
- Период: 01.01.2017 - 26.06.2017
- Данные SCADA + журнал остановок

## Что делаю в проекте:
1. Загрузка и предобработка данных
2. Анализ пропусков
3. Объединение данных датчиков и событий
4. Разведочный анализ

## Технологии
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
